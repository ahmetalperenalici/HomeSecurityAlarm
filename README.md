# Home Security Alarm System

Arduino-based home security system with:
- PIR sensor for motion detection
- Arming delay (5 seconds)
- Siren effect with LED and buzzer
- Button debounce for reliable control
- LCD display showing system status

## Hardware Required
- Arduino board
- PIR sensor
- 2 LEDs (Red and Green)
- Buzzer
- 2 Push buttons
- LCD 16x2 with I2C

## Usage
- Press ARM button to arm the system (5 sec delay)
- Press DISARM button to deactivate
- PIR sensor triggers the alarm, LED and buzzer activate with siren pattern
